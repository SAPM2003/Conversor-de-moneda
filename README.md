# Conversor-de-moneda# 
Conversor de Moneda en Java  
Aplicación de consola desarrollada en **Java 24** que permite convertir entre distintas monedas utilizando la API pública **ExchangeRate-API**.  
El usuario puede elegir la moneda origen, la moneda destino y la cantidad a convertir.
---

## Funcionalidades

- Conversión entre 6 monedas:
  - USD → EUR
  - EUR → USD
  - USD → GBP
  - GBP → USD
  - USD → JPY
  - JPY → USD
- El usuario puede ingresar cantidades con punto o coma decimal.
- Menú interactivo que permite realizar múltiples conversiones.
- Opción para salir del programa.
- Obtiene tasas de cambio en tiempo real desde **ExchangeRate-API**.
- Formatea los resultados con dos decimales.
